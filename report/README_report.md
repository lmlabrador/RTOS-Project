# TickFlow – Deliverable Files

## Files
| File | Purpose |
|---|---|
| `TickFlow_Report.tex` | IEEE conference paper (LaTeX source) |
| `TickFlow_Presentation.html` | Slide deck (open in browser) |

## Compiling the Report to PDF

### Option A – MiKTeX / TeX Live (local)
```bash
# Run twice to resolve cross-references
pdflatex TickFlow_Report.tex
pdflatex TickFlow_Report.tex
```

### Option B – Overleaf (recommended, no install)
1. Go to overleaf.com → New Project → Upload Project
2. Upload `TickFlow_Report.tex`
3. Set compiler to **pdfLaTeX**
4. Click Compile → Download PDF

## Viewing the Presentation
Open `TickFlow_Presentation.html` in Chrome, Edge, or Firefox.
- Arrow keys / Space → navigate slides
- Bottom buttons also work
- Slides 14–15 are backup Q&A slides
